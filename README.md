**DESCRIPCIÓN**
Este proyecto es una aplicación de consola en Java que gestiona diversos tipos de contenidos audiovisuales (Películas, Series, Documentales, etc.). El objetivo es aplicar los conceptos de POO.

Conceptos Aplicados
**Herencia y Polimorfismo**: Uso de clases abstractas y métodos sobreescritos.
**Relaciones entre Clases**: Implementación de Agregación, Composición y Asociación.
**Encapsulamiento**: Uso de modificadores de acceso privados con métodos Getter y Setter.

**ESTRUCTURA**
El código se organiza en paquetes:
uni1a: Contiene la lógica de negocio y las clases del dominio.
poo: Contiene la clase de prueba principal.
